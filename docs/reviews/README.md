# Review Notes

Store durable review artifacts here with the same lifecycle split as execution plans.

## Directories

- `active/`: review notes for work that is still being addressed
- `archive/`: closed, superseded, or historical review notes

## Naming

Use date-prefixed names:

- `2026-03-09-workproof-review.md`
- `2026-03-09-wage-shield-security-review.md`

## Content

Prefer concise documents that separate:

- findings
- open questions
- residual risks
- deferred follow-up work

When a review corresponds to an execution plan, link them to each other.
